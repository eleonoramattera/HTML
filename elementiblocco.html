Tag di heading

Sono utilizzati per definire i livelli di titolo o intestazione all'interno di un documento. Questi tag vanno dall'heading level 1 (<h1>) al heading level 6 (<h6>), dove <h1> è il titolo più importante e <h6> è il meno importante. <h1> </h1>
<h2> </h2>
<h3> </h3>
<h4> </h4>
<h5> </h5>
<h6> </h6>

Paragrafi

Questo tag è fondamentale per strutturare e organizzare il contenuto di una pagina web in blocchi di testo separati. Crea un nuovo blocco di testo inserendo un margine prima e dopo il testo, creando un aspetto distintivo per ogni paragrafo. È importante utilizzare il tag correttamente all'interno della struttura del tuo documento HTML, in modo da rendere il contenuto ben organizzato. Un documento HTML ben strutturato e con l'uso corretto dei tag di paragrafo può migliorare l'indicizzazione da parte dei motori di ricerca e l'esperienza di lettura da parte degli utenti.
<p>questo è un paragrafo</p>
<p>questo è un altro paragrafo</p>

Contenitore generico / division

E' un elemento di contenitore generico che viene utilizzato per raggruppare e organizzare altri elementi HTML in blocchi distinti. La funzione principale è quindi suddividere il contenuto della pagina in diverse parti o sezioni. Non ha alcun significato semantico specifico e non altera il significato o la struttura del contenuto che contiene. Invece, viene utilizzato per scopi di layout e stile, consentendo agli sviluppatori di raggruppare insieme elementi correlati o di creare contenuti strutturati. Ad esempio, prima dell'avvento di HTML5 il div veniva usato in questo modo <div class="header"> </div> per creare un contenitore di intestazione 

              
                <div>
                   <p>primo paragrafo del primo div </p>
                   <p>secondo paragrafo del primo div </p>
                </div>
                <div>
                   <p>primo paragrafo del secondo div </p>
                   <p>secondo paragrafo del secondo div </p>
                </div>

Citazioni

E' utilizzato per citare il contenuto di un altro autore o una fonte esterna all'interno di un documento. Questo tag contiene alcune proprietà css di default come l'indentazione sui margini sinistro e destro, che lo distingue dal resto del testo. Questa formattazione visiva aiuta a rendere chiaro che il contenuto è una citazione. 

   
                    
                        <blockquote>
                         La vera grandezza non consiste in ciò che hai, ma in ciò che sei.    
                        </blockquote>

Liste di definizioni

Le liste di definizioni in HTML sono create utilizzando il tag <dl>, che sta per "Description List" (lista di definizioni). Questo tipo di lista è utilizzato per presentare una serie di termini (definizioni) e le relative definizioni (descrizioni) in modo strutturato e semanticamente significativo. All'interno del tag <dl>, utilizziamo i tag <dt>(Definition Term) per definire i termini (definizioni) e i tag <dd> (Definition Description) per fornire le descrizioni di ciascun termine. 

                    
                       <dl>                      
                          <dt> HTML </dt>
                          <dd>Linguaggio di markup utilizzato per creare pagine web. </dd>
                          
                          <dt>CSS </dt>
                          <dd> Linguaggio utilizzato per lo stile delle pagine web. </dd>
                        </dl>


Output:
HTML
Linguaggio di markup utilizzato per creare pagine web.
CSS
Linguaggio utilizzato per lo stile delle pagine web.

Liste ordinate

Le liste ordinate in HTML sono create utilizzando il tag <ol>, che sta per "Ordered List". Questo tag è utilizzato per creare elenchi numerati o elenchi ordinati, in cui ogni elemento della lista è preceduto da un numero sequenziale. All'interno del tag <ol>, utilizziamo il tag <li> (List Item) per definire ogni elemento della lista. Ogni <li> contiene il testo o il contenuto che vuoi mostrare nella lista. L'elenco sarà numerato automaticamente dall'1 in poi,ma personalizzare l'aspetto delle liste ordinate cambiarndo il tipo di numerazione (come numeri, lettere, numeri romani, ecc.) attraverso attributo type. 

                        
                            <ol>
                            <li> primo elemento </li>
                            <li>  secodno elemento</li>
                            </ol>
Output:
  1. primo elemento
  2. secondo elemento
                        
                            <ol type="A"> 
                            <li> primo elemento </li>
                            <li>  secodno elemento</li>
                            </ol>
Output:
  1. primo elemento
  2. secondo elemento

Liste non ordinate

Le liste non ordinate in HTML sono create utilizzando il tag <ul>, che sta per "Unordered List" (lista non ordinata). Questo tag è utilizzato per creare elenchi puntati o elenchi non numerati, in cui ogni elemento della lista è preceduto da un simbolo di puntino, pallino, o altro simbolo. All'interno del tag <ul>, utilizziamo il tag <li>(List Item) per definire ogni elemento della lista. Ogni <li> contiene il testo o il contenuto che vuoi mostrare nella lista. L'elenco sarà puntato automaticamente con il simbolo predefinito, di solito un puntino ma è possibile personalizzare l'aspetto delle liste non ordinateutilizzando CSS 

                    
                        <ul>
                        <li> primo elemento </li>
                        <li>  secodno elemento</li>
                        </ul>
Output:
  • primo elemento
  • secondo elemento
                    
                        <ul style="list-style-type: square;">
                        <li> primo elemento </li>
                        <li>  secodno elemento</li>
                        </ul>
Output:
  • primo elemento
  • secondo elemento
